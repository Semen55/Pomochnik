!! для создания воркспейса введите название воркспейса, потом введите путь и нажмите добавить, для добавления сайта/программы
не меняя имени воркспейса очистите поле с адресом сайта/путём программы и пропишите по новой, после чего нажмите добавить.

команды для кейсона:
        узнать время: ("скажи текущее время", "сколько сейчас времени", "который час"),
        поиск файла: ("найди файл", "открой файл", "мне нужен файл"),
        создание папки: ("создай папку", "сделай папку"),
        использование воркспейса: ("активируй воркспэйс", "воркспэйс","запусти воркспейс"),
        создание воркспейса: ("создай воркспэйс", "добавь воркспэйс", "мне нужен новый воркспейс"),
        поиск в интернете: ("что ты знаешь про ...", "что такое ...")

!!! для запуска кейсона перейдите в папку dist-caison-caison.exe
